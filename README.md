# Nagios-Monitoring
---

 Nagios Monitoring Setup with CentOS Server and Fedora Client/s
 --------------------------------------------------------------
1. Install Apache web server
2. Download Nagios
3. Download necessary Script(init,commandmode,config,webconfig)
4. Download & install Nagios Plugins
5. Install nrpe plugin & config
6. Edit config files
7. Download OpenSSL file
8.  install xinetd & config
9. Monitor Client(Fedora machine) Using nrpe in Nagios Dashboard


DNS Caching Setup with CentOS Server and Fedora Clients
--------------------------------------------------------

1. Install & Config DNS server
2. Edit resolve.cfg file
3. config Forward & Reverse Zones
4. Check query time
